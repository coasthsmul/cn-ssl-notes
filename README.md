# cn-ssl-notes

本仓库用于归档与发布一系列独立 HTML 页面，内容主要围绕 SSL/TLS 相关技术笔记与参考资料。这些页面均为独立构建，不针对任何特定域名或网站，仅作为知识整理与共享用途。

## 项目简介

`cn-ssl-notes` 是一个轻量级的文档归档项目，旨在以 HTML 页面的形式整理和呈现 SSL/TLS 相关的学习笔记、配置参考、常见问题等内容。每个页面均为自包含的 HTML 文件，可直接在浏览器中打开阅读，无需额外依赖。

## 目录说明

```
cn-ssl-notes/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、图片等）
├── README.md       # 本文件
└── index.html      # 可选的入口页面（如有）
```

- **pages/**：主要归档目录，每个 `.html` 文件对应一份独立笔记或参考文档。
- **assets/**：存放页面所需的样式、脚本或图片等资源文件，保持页面可离线使用。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部服务或动态内容。
- 页面内容侧重于技术笔记与经验总结，不包含任何商业推广或诱导点击元素。
- 归档内容会不定期更新，新增或修订页面将直接提交至 `pages/` 目录。

## 维护说明

- 欢迎通过 Issue 或 Pull Request 提交建议、修正或补充内容。
- 提交新页面时，请确保文件为纯 HTML 格式，并尽量保持风格一致。
- 请勿在页面中添加任何指向特定域名的链接或营销性内容。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可阅读。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。
